My Projects

Sound Off - 2021

My love of music and interest to explore new technology led to the development of this project. There are two main technologies that drive this app, Spotify Music API and SocketIO realtime socket communication. The Spotify API is what is used to authenticate users, pull their playlist info, and to play music in the web player. The Socket IO package gives the app the ability to communicate in realtime with concurrent users, and this allows for the creation of separate listening rooms with live chat for concurrent users in a room. This app has only just been created and still has much potential!

Troves - 2020

This was a location-based mobile app built with an idea to use crowd-sourced points of interest to create connections between people and their community. The user could add local points of interest to the public or just to private groups of friends. Built on a react-native stack, the app was designed to work cross-platform. This was my first real venture into mobile development, and using google API, react-native map packages, and other packages to provide a synchronized service between members turned out to be a great challenge and learning experience!
